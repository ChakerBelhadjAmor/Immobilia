# Raw Legal Sources

Place original legal source files here. Keep the source category clear because
the ingestion pipeline uses it for metadata filtering.

Examples:

```text
legifrance/loi_1989_baux_habitation.pdf
service_public/diagnostics_immobiliers_obligatoires.pdf
anil/location_logement_vide.pdf
notaires_france/compromis_de_vente.pdf
```
